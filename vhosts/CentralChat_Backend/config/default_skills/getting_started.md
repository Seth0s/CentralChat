# Getting Started with Central

Central is your personal AI assistant running on your own infrastructure.
You control your data, your models, and your agents.

## Core principles

- **You are in control.** Every action that modifies your system requires
  your explicit approval (HITL).
- **1 file = 1 domain.** Central is optimised for AI agents to understand
  your project quickly.
- **Persistent memory.** Central remembers across sessions via `memory` tool.
- **Default tools.** 12 built-in tools let Central read, write, search,
  execute code, delegate tasks, and more.

## Quick commands

- `read_file` — read any file with line numbers
- `write_file` — create or overwrite a file
- `terminal` — run shell commands (requires approval)
- `search_files` — grep for patterns in your project
- `memory` — save facts to persistent memory
- `web_search` — search the web via DuckDuckGo

## Getting help

Ask Central anything about your project. It will read files,
search code, and remember context across sessions.
